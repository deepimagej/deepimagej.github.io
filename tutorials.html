How to run a model in deepImageJ?
  • Click on DeepImageJ Install, and install one of the suggested models compatible with deepImageJ from the BioImage Model Zoo list. There's no need to restart Fiji once the model is installed.
  • Once the model is installed (the installing pop up window closes), you can close the window.
  • Click on DeepImageJ Run. Choose the recently installed model and click on Run on Test Image. To process and image, follow the same steps but click on Accept rather than on Run on a test image.

DeepImageJ Run is macro recordable!
  • Start the macro recorder.
  • Use DeepImageJ Run on an image.
  • Copy the code line given by the macro recorder.
  • Create a new ImageJ macro, paste the line and run it.
